Algumas dúvidas sobre a Calculadora de Notas

Como funciona o site?

Inicialmente, você deve digitar suas notas. É necessário inserir pelo menos uma nota para que o sistema calcule sua média e indique sua situação. O sistema aceita apenas notas iguais ou superiores a 0 (zero) e iguais ou inferiores a 10 (dez).

O sistema pode me mostrar quanto eu preciso tirar nas proximas notas para atingir uma média aprovativa?

Sim, o sistema é capaz de mostrar quanto você precisa tirar nas próximas notas para atingir uma média aprovativa. Ele verifica as notas que você forneceu e calcula qual nota você precisa obter nas próximas avaliações para alcançar a média desejada.

O sistema me mostra se eu já estou de recuperação final?

Sim, o sistema é capaz de verificar se você já está em recuperação final. Se você forneceu duas notas e essas notas são muito baixas, mesmo que você tire 10 nas notas não digitadas, você já está em situação de recuperação final. O sistema faz essa análise e informa sobre a necessidade de recuperação com base nas notas fornecidas.

O sistema mostra quanto eu preciso tirar na prova final para ser aprovado(a)?

Sim, se você fornecer as quatro notas, o sistema calculará a média. Se a média não for aprovativa, o sistema mostrará quanto você precisa tirar na prova final para ser aprovado.
